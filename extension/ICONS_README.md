# Icônes de l'extension

L'extension nécessite 3 fichiers d'icônes PNG :

- `icon16.png` : 16x16 pixels
- `icon48.png` : 48x48 pixels  
- `icon128.png` : 128x128 pixels

## Création des icônes

Vous pouvez :
1. Utiliser le logo Etsmart existant et le redimensionner
2. Créer des icônes simples avec un outil comme Figma, Canva, ou un générateur d'icônes
3. Utiliser un placeholder temporaire pour le développement

## Placeholder temporaire

Pour tester rapidement, vous pouvez créer des fichiers PNG simples avec une couleur de fond et le texte "E" ou utiliser le logo Etsmart.
