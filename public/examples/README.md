# Exemple de screenshot

## Fichier attendu

Placez votre image d'exemple de screenshot AliExpress dans ce dossier avec le nom :

**`screenshot-example.png`**

## Spécifications

- **Format** : PNG
- **Résolution** : Suffisante pour une lecture claire (minimum desktop)
- **Taille** : Optimisée, < 500 KB recommandé
- **Contenu** : Page produit AliExpress avec le produit "Etsmart Cup" (mug blanc avec logo Etsmart)

## Description de l'image attendue

L'image doit afficher clairement :
- Une page produit AliExpress
- Un produit unique et centré (Etsmart Cup)
- Une image produit principale visible
- Un titre produit lisible ("Etsmart Cup")
- Un prix visible
- Une mise en page réaliste AliExpress

## Utilisation

Cette image est affichée sur la page d'import produit (`/app`) pour servir d'exemple visuel aux utilisateurs. Elle est purement illustrative et ne déclenche aucune analyse.
